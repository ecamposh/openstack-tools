pip install python-openstackclient

Create configuration file

C:\Users\user\.config\openstack\clouds.yaml


clouds:
  <cloud_name>:
    auth_type: password
    auth:
      auth_url: "https://keystone.api.<region>.rackspacecloud.com/v3/"
      username:
      password:
      project_domain_name: rackspace_cloud_domain
      user_domain_name: rackspace_cloud_domain



Test it


openstack --os-cloud <cloud_name> --os-username <username> --os-password <legacy_cloud_api_key> --os-project-name <project_name> server list
